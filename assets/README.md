# Static assets

| File | Purpose |
|------|--------|
| **`cv.pdf`** | Your CV; replace this file and push to update the site download. |
| **`profile.jpg`** | Home page portrait (not committed if you add it locally only—commit it if the live site should show it). |
| **`og-image.png`** | Default Open Graph / social preview image (1200×630 recommended). |

The repository includes a **placeholder** `cv.pdf` and `og-image.png` so builds succeed; swap them for your real files when ready.
